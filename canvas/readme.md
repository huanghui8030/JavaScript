## 简介

### canvas是什么
是HTML5中重要的元素,和audio、video元素类似完全不需要任何外部插件就能够运行.

Canvas中文翻译就是”画布”.它提供了强大的图形的处理功能(绘制,变换,像素处理…)。

但是需要注意，canvas 元素本身并不绘制图形，它只是相当于一张空画布。

如果开发者需要向 canvas 上绘制图形，则必须使用 JavaScript 脚本进行绘制。

### 特点
- 基础图形的绘制
- 文字的绘制
- 图形的变形和图片的合成
- 图片和视频的处理
- 动画的实现
- 小游戏的制作

### 浏览器支持
- IE9+，Firefox、chrome、Safari、Opera
